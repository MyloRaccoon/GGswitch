# GGswitch
Godot Global switch manager plugin
Like RPG Maker switches but in a Godot Panel

<img width="286" height="479" alt="image" src="https://github.com/user-attachments/assets/ea2c41fe-0a6a-4754-90ef-6905d1b25e4e" />


## features
- Declare switches with the ui
- Set switches' default value with the ui
- Delete switches with the ui
- Check if a switch exists with GGswitch.get_switch()
- Switch a switch with GGswitch.switch()
